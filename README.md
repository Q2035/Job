### 用户

#### 登录

使用用户名和密码进行登录。

管理员和普通用户所对应的页面不一致

系统管理员设置普通管理员，普通管理员可为指定班级发布作业。

> 系统管理员可以创建新的班级，可以指定普通管理员
>
> 一个普通管理员可对应多个班级，可以对已存在的班级进行人员导入，但是无法创建新的班级

#### 导入

1. 管理员登录后选择新建班级导入用户
2. 选择已存在的班级导入用户。

#### 上传

1. 用户上传后可以下载上传后的文件或者在线查看文件
2. 可进行覆盖上传

#### 通知

1. 用户可以设置是否接收邮件提醒
2. 管理员发布作业时可以设置是否启用邮件提醒

> 来一张表存放所有不愿接收提醒的用户
